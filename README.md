# Blood-Bank-&-Donation-Management-System

Blood Bank Donation System is a php based web project with both admin and user layouts.

# Installation

1. Install XAMPP or WAMPP.

2. Open XAMPP Control panal and start [apache] and [mysql] .

3. Download project from github(https://github.com/Abinash-01/Blood-Bank-and-Donation-Management-System.git).

   OR follow gitbash commands <br>
     ```t
         i>cd C:\\xampp\htdocs\
        ii>git clone https://github.com/Abinash-01/Blood-Bank-and-Donation-Management-System.git.git
     
     
4. Extract files in C:\xampp\htdocs.

5. Open link localhost/phpmyadmin

6. Click on new at side navbar.

7. Give a database name as (blood_bank_database) hit on create button.

8. After creating database name click on import.

9. Browse the file in directory[BDMS/sql/blood_bank_database.sql].

10. After importing successfully.

11. Open any browser and type http://localhost//home.php to open User layout.
     
# Admin Panel
   Open any browser and type http://localhost/Blood-Bank-And-Donation-Management-System/admin/login.php to open Admin Login Panel.
